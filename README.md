# Internship at IIT Kanpur - Project Repository

## Overview
This repository contains the code for four projects completed during the internship at IIT Kanpur. Each project is described below, including a brief explanation and the code associated with it.

### Project 1: Word Guessing Game
- Description: This project is a word guessing game with three difficulty levels: Easy, Standard, and Hard. The user selects a difficulty level, and the program randomly selects a word for the user to guess. The user has ten attempts to guess the word correctly.

### Project 2: Tic-Tac-Toe Game
- Description: This project is a console-based Tic-Tac-Toe game where two players take turns to make their moves. The program checks for a winner after each move and announces the winner or declares a draw at the end.

### Project 3: Age Calculator
- Description: This project calculates the age of a person based on their date of birth. The user is prompted to input their date of birth, and the program outputs the age in years, months, and days.

### Project 4: Employee Management System
- Description: This project is an Employee Management System with functionalities like inserting, editing, displaying, and deleting employee records. The data is stored in a file, and the program provides a simple command-line interface for user interaction.

### How to Run
To run any of the projects, make sure you have Python installed on your machine. Simply copy the code for the desired project into a Python environment and execute the script.

Feel free to explore each project's code and modify it as needed. If you have any questions or suggestions, please don't hesitate to reach out.
